智能提取API

文档更新时间：2019-08-30
注意
  
1、所有API的调用，必须附带"token"这个参数，即公开的AppKey；
2、API返回的JSON格式数据是经过UTF-8编码的，这个文档里的"成功返回示例"为了直观，已经decode过；
3、由于数据源方面还有服务器负载过大的原因，API中的值有时返回可能为空或者等待时间稍久；
4、本接口由于需要用到一些关于机器学习的算法，所以有时需要等待的时间稍长。
  

数据格式说明：
API返回的数据包括了以下内容：



1、商品价格历史变化查询


	支持天猫(detail.tmall.com、detail.m.tmall.com)、淘宝(item.taobao.com、h5.m.taobao.com)、
        京东(item.jd.com、item.m.jd.com)、一号店(item.yhd.com）、苏宁易购(product.suning.com)、
        网易考拉(goods.kaola.com)、当当网(product.dangdang.com)、亚马逊中国(www.amazon.cn)、国美(item.gome.com.cn)等电商
        商品详情的历史价格查询。



  
    
    
  
    

      地址
      http://scenicmonitor.top/interface/api/getGoodsPrice?
    

    

      方法
      GET
    

    

      参数
      
* url：商品链接
* token：Appkey，必选参数
      
    

    

      返回 
      
数据格式 {
	
	data: [["2019,7,7", "19"], ["2019,7,8", "19"], ["2019,7,9", "19"], ["2019,7,10", "19"], ["2019,7,11", "19"],…]
}

      
    

  


请求示例：
http://scenicmonitor.top/interface/api/getGoodsPrice?token=bGlhbnpvbmdzaGVuZw==
成功返回示例：
data: [["2019,7,7", "19"], ["2019,7,8", "19"], ["2019,7,9", "19"], ["2019,7,10", "19"], ["2019,7,11", "19"],…]
可能的错误返回信息：
{"status": 0, "message": "appkey错误"}


2、获取手机系统历史占有率



  
    
    
  
    

      地址
      http://scenicmonitor.top/interface/api/getMobileSystemShare?
    

    

      方法
      GET
    

    

      参数
      
* token：Appkey，必选参数
      
    

    

      返回 
      
数据格式 {
	
	android: [{pid__version: "Android+4.3", ddate: "2014-01-01", rate: 4.91},…]
	apple: [{pid__version: "OS 6.1", ddate: "2014-01-01", rate: 3.09},…]
}
* android：安卓
* apple：苹果
* pid__version：手机系统版本
* ddate：日期
* rate：占有率
      
    

  


请求示例：
http://scenicmonitor.top/interface/api/getMobileSystemShare?token=bGlhbnpvbmdzaGVuZw==
成功返回示例：
{
android: [{pid__version: "Android+4.3", ddate: "2014-01-01", rate: 4.91},…]
apple: [{pid__version: "OS 6.1", ddate: "2014-01-01", rate: 3.09},…]
}
可能的错误返回信息：
{"status": 0, "message": "appkey错误"}


3、获取手机网络运营商历史占有率



  
    
    
  
    

      地址
      http://scenicmonitor.top/interface/api/getOperatorShare?
    

    

      方法
      GET
    

    

      参数
      
* token：Appkey，必选参数
      
    

    

      返回 
      
数据格式 {
	
	share: [{pid__name: "中国移动", ddate: "2014-01-01", rate: 59.83},…]
}

* pid__name：运营商
* ddate：日期
* rate：占有率
      
    

  


请求示例：
http://scenicmonitor.top/interface/api/getOperatorShare?token=bGlhbnpvbmdzaGVuZw==
成功返回示例：
{
share: [{pid__name: "中国移动", ddate: "2014-01-01", rate: 59.83},…]
}
可能的错误返回信息：
{"status": 0, "message": "appkey错误"}
