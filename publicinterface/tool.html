智能提取API

文档更新时间：2019-08-30
注意
  
1、所有API的调用，必须附带"token"这个参数，即公开的AppKey；
2、API返回的JSON格式数据是经过UTF-8编码的，这个文档里的"成功返回示例"为了直观，已经decode过；
3、由于数据源方面还有服务器负载过大的原因，API中的值有时返回可能为空或者等待时间稍久；
4、本接口由于需要用到一些关于机器学习的算法，所以有时需要等待的时间稍长。

1、商品价格历史变化查询


	支持天猫(detail.tmall.com、detail.m.tmall.com)、淘宝(item.taobao.com、h5.m.taobao.com)、
        京东(item.jd.com、item.m.jd.com)、一号店(item.yhd.com）、苏宁易购(product.suning.com)、
        网易考拉(goods.kaola.com)、当当网(product.dangdang.com)、亚马逊中国(www.amazon.cn)、国美(item.gome.com.cn)等电商
        商品详情的历史价格查询。



  
    
    
  
    

      地址
      http://scenicmonitor.top/interface/api/getGoodsPrice?
    

    

      方法
      POST
    

    

      参数
      
* url：商品链接，必选参数
* token：Appkey，必选参数
      
    

    

      返回 
      
数据格式 {
	
	data: [["2019,7,7", "19"], ["2019,7,8", "19"], ["2019,7,9", "19"], ["2019,7,10", "19"], ["2019,7,11", "19"],…]
}

      
    

  


请求示例：
http://scenicmonitor.top/interface/api/getGoodsPrice?token=bGlhbnpvbmdzaGVuZw==&url=https://chaoshi.detail.tmall.com/item.htm?id=556442478145&spm=875.7931836/B.2017039.7.66144265uFeYac&scm=1007.12144.81309.73136_0_0&pvid=3bd567dd-9892-41bd-b5c5-49e748c2d1e1&utparam=%7B%22x_hestia_source%22%3A%2273136%22%2C%22x_object_type%22%3A%22item%22%2C%22x_mt%22%3A8%2C%22x_src%22%3A%2273136%22%2C%22x_pos%22%3A4%2C%22x_pvid%22%3A%223bd567dd-9892-41bd-b5c5-49e748c2d1e1%22%2C%22x_object_id%22%3A556442478145%7D
成功返回示例：
data: [["2019,7,7", "19"], ["2019,7,8", "19"], ["2019,7,9", "19"], ["2019,7,10", "19"], ["2019,7,11", "19"],…]
可能的错误返回信息：
{"status": 0, "message": "appkey错误"}


2、文本关键词提取


    支持微博、今日头条、微信等大中型网站文本链接解析



  
    
    
  
    

      地址
      http://scenicmonitor.top/interface/api/getUrlKeyword?
    

    

      方法
      POST
    

    

      参数
      
* url：文本链接，必选参数
* allowPos : 词性代码，必选参数
* token：Appkey，必选参数
      
    

    

      返回 
      
数据格式 {
	data:[('杜特', 1.0), ('王毅', 0.7383340036343697), ('杨洁篪', 0.7355249880641205), ('何立峰', 0.7323975331449243), ('祝贺', 0.3742371835280207)]

}

      
    

  


请求示例：
http://scenicmonitor.top/interface/api/getUrlKeyword?token=bGlhbnpvbmdzaGVuZw==&allowPos=nr&url=https://www.toutiao.com/a6730603909976424963/
成功返回示例：
{
data:[('杜特', 1.0), ('王毅', 0.7383340036343697), ('杨洁篪', 0.7355249880641205), ('何立峰', 0.7323975331449243), ('祝贺', 0.3742371835280207)]

}
可能的错误返回信息：
{"status": 0, "message": "appkey错误"}
