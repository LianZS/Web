终端数据API

文档更新时间：2019-08-30
注意
  
1、所有API的调用，必须附带"token"这个参数，即公开的AppKey；
2、API返回的JSON格式数据是经过UTF-8编码的，这个文档里的"成功返回示例"为了直观，已经decode过；
3、由于数据源方面还有服务器负载过大的原因，API中的值有时返回可能为空或者等待时间稍久；
4、所有终端数据是通过爬虫获取，若有欠缺，敬请原谅
  

数据格式说明：
API返回的数据包括了以下内容：



  
    
    
  
    

      字段
      字段说明
    

  
  
    

      pid__name
      品牌/运营商/网络
    

      

      ddate
      日期
    

    

      rate
      国内市场占有率
    

        

      pid__version
      手机系统版本
    

        

      rate
      国内市场占有率
    

 

  




1、获取手机品牌历史占有率



  
    
    
  
    

      地址
      http://scenicmonitor.top/interface/api/getBrandShare?
    

    

      方法
      GET
    

    

      参数
      
* token：Appkey，必选参数
      
    

    

      返回 
      
数据格式 {
	
	share: [{pid__name: "华为", ddate: "2014-01-01", rate: 1.14},…]
}
* pid__name：品牌
* ddate：日期
* rate：占有率
      
    

  


请求示例：
http://scenicmonitor.top/interface/api/getBrandShare?token=bGlhbnpvbmdzaGVuZw==
成功返回示例：
share: [{pid__name: "华为", ddate: "2014-01-01", rate: 1.14},…]

可能的错误返回信息：
{"status": 0, "message": "appkey错误"}


2、获取手机系统历史占有率



  
    
    
  
    

      地址
      http://scenicmonitor.top/interface/api/getMobileSystemShare?
    

    

      方法
      GET
    

    

      参数
      
* token：Appkey，必选参数
      
    

    

      返回 
      
数据格式 {
	
	android: [{pid__version: "Android+4.3", ddate: "2014-01-01", rate: 4.91},…]
	apple: [{pid__version: "OS 6.1", ddate: "2014-01-01", rate: 3.09},…]
}
* android：安卓
* apple：苹果
* pid__version：手机系统版本
* ddate：日期
* rate：占有率
      
    

  


请求示例：
http://scenicmonitor.top/interface/api/getMobileSystemShare?token=bGlhbnpvbmdzaGVuZw==
成功返回示例：
{
android: [{pid__version: "Android+4.3", ddate: "2014-01-01", rate: 4.91},…]
apple: [{pid__version: "OS 6.1", ddate: "2014-01-01", rate: 3.09},…]
}
可能的错误返回信息：
{"status": 0, "message": "appkey错误"}
